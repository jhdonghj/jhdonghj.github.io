Bayes’ Nets

Bayes’ nets: a technique for describing complex joint distributions (models) using simple, local distributions (conditional probabilities)

Semantics

Bayes’ nets consist of:

  • A directed, acyclic graph of nodes, one per variable $X$
  • A conditional distribution for each node $P(X|A_1\dots A_n)$, where $A_i$ is the $i^{th}$ parent of $X$, sorted as a conditional probability table (CPT). Each CPT has $n+2$ columns: $n$ parent values, 1 child value, and 1 probability value.

Example

In this Bayes Net, we would store probability tables $P(B),P(E),P(A|B,E),P(J|A)$ and $P(M|A)$.

Structure of Bayes Nets

  • Each node is conditionally independent of all its ancestor nodes (non-descendants) in the graph, given all of its parents.
    • 在已知父节点状态的情况下，节点与其祖先节点独立
  • Each node is conditionally independent of all other variables given its Markov blanket. A variable’s Markov blanket consists of parents, children, children’s other parents

Size of Bayes’ Nets

Size of a joint distribution over N boolean variables: $2^N$

Size of a N-node net with up to k parents per node: $O(N\cdot 2^{k+1})$

Bayes’ nets save a huge space.

General Case, and D-separation

General problem:

Given a Bayes Net $G$, two nodes $X$ and $Y$, and a (possibly empty) set of nodes $\{Z_1,\dots,Z_k\}$ that are observed, is $X$ independent of $Y$ under given conditions?

D-separation algorithm:

  1. Shade all observed nodes $\{Z_1,\dots,Z_k\}$ in the graph.
  2. Enumerate all paths from $X$ to $Y$.
  3. For each path:
    1. Decompose the path into triples (segments of 3 nodes).
    2. If all triples are active, this path is active and d-connects X to Y.
  4. If no path d-connects X to Y, then X and Y are d-separated, so they are conditionally independent given $\{Z_1,\dots,Z_k\}$.

Inference

Inference: calculating some useful quantity from a joint probability distribution. In other words, finding the value of some probability distribution $P(Q_1\dots Q_k|e_1\dots e_k)$.

Variable Elimination

Factor

A factor is defined simply as an unnormalized probability.

$P(Y_1\dots Y_n|X_1\dots X_m)$ is a factor. Its values are $P(y_1\dots y_n|x_1\dots x_m)$.

一个P的所有可能取值的表就是factor。

Join Factors

Join two factors into one factor based on common variables.

Eliminate Variables

Take a factor and sum out a variable. Shrinks a factor to a smaller one.

Multiple Join, Multiple Eliminate (= Inference by Enumeration)

Marginalizing Early (= Variable Elimination)

Evidence

General Variable Elimination

Query: $P(Q|E_1=e_1,\dots,E_k=e_k)$.

  1. Start with initial factors (CPTs)
  2. While there are hidden variables:
    1. Select a hidden variable $X$.
    2. Find all factors that mention $X$.
    3. Join these factors.
    4. Eliminate $X$ by summing out.
  3. Join all remaining factors and normalize.

Complexity

Order of elimination matters a lot.

Solving VE can be reduced to solving 3-SAT problem, so it is NP-hard.

On polytrees(a directed acyclic graph, DAG), VE is linear in the size of the network.

Approximate Inference: Sampling

An alternate approach for probabilistic reasoning is to implicitly calculate the probabilities for our query by simply counting samples.

Downside: may sample many irrelevant variables. If we want to compute $P(C|-t)$, we’d have to throw away 99% of our samples.

Rejection Sampling

Early rejection of samples that don’t match evidence.

For example, for the query $P(C|-t)$, we’d avoid generating a value for C unless t is false.

Likelihood Weighting

Idea: fix evidence variables and sample the rest

Sample with a parameter $w$ storing the weight of the sample. Change $w$ when evidence is going to be sampled.

Weighted sampling distribution is consistent.

In likelihood weighting, evidence influences the choice of downstream variables, but not upstream ones. It leads to Gibbs sampling.

Gibbs Sampling

Steps:

  1. Set all variables(except evidence variable) to some totally random value. Set evidence variable to the given value.
  2. Repeatedly pick one variable at a time, clear its value, and resample it given the values currently assigned to all other variables.

Conclution

Bayes’ Nets is a powerful representation of joint probability distributions. Its topological structure encodes independence and conditional independence relationships, and we can use it to model arbitrary distributions to perform inference and sampling.
